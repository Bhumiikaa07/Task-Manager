# Task Manager Application

A responsive and interactive Task Manager web application built using HTML, CSS, and JavaScript to help users efficiently organize and manage daily tasks.

---

## Features

- Add new tasks
- Delete existing tasks
- Mark tasks as completed
- Persistent task storage using Local Storage
- Responsive and user-friendly interface
- Dynamic UI updates using DOM Manipulation

---

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Browser Local Storage

---

## Project Overview

The Task Manager Application is a frontend-based web project designed to improve productivity and task organization. Users can create, manage, complete, and remove tasks dynamically without reloading the webpage.

The application uses JavaScript DOM Manipulation for real-time UI updates and Browser Local Storage to persist task data even after refreshing or reopening the browser.

- CRUD Operations
- DOM Manipulation

- Local Storage
